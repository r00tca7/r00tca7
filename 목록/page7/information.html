메모 목록 페이지 설명

기능 개요

이 페이지는 사용자가 작성한 메모를 목록으로 보여주고, 메모를 추가, 수정, 삭제할 수 있는 기능을 제공합니다.

주요 기능

  1. 메모 추가: "새 메모 추가" 버튼을 클릭하면 새로운 메모를 작성할 수 있는 페이지가 새 탭에서 열립니다.
  2. 메모 목록 표시: 로컬 스토리지에 저장된 메모를 불러와 목록 형태로 표시합니다. 메모는 제목과 내용을 포함합니다.
  3. 메모 수정 및 삭제: 각 메모 옆에 있는 "수정" 버튼을 클릭하면 해당 메모를 수정할 수 있는 페이지가 열리며, "삭제" 버튼을 클릭하면 메모가 삭제됩니다.
  4. 페이지네이션: 메모가 많을 경우, 페이지를 나누어 표시하여 사용자가 쉽게 탐색할 수 있도록 합니다.

구성 요소

페이지는 다음과 같은 요소들로 구성되어 있습니다:

  • 헤더: 페이지 제목 "메모 목록"을 중앙에 표시합니다.
  • 메모 목록: 로컬 스토리지에서 가져온 메모들이 표시되는 영역입니다.
  • 페이지네이션: 메모가 여러 페이지로 나누어질 때 페이지 번호를 표시하는 버튼들이 위치합니다.
